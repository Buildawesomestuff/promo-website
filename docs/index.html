Prepared courses and sample programs

For developers the course helps to improve their software development skills and qualification in Web Technologies, as well as systematise their knowledge in programming in general. After the course developers can apply new ideas from a variety of frameworks on their daily job. This helps to write more stable and less error-prone software, improves and speeds up development process and experience.

The Modern JavaScript Frameworks course
  • extends JavaScript frameworks and libraries key principles and features knowledge
  • provides an overview of software design patterns, such as Event Loop, Flux, Immutable, Virtual DOM, Dependency Injection, Observables, MVC
  • exposes standards of projecting and managing, building and testing server and client applications
The course embraces
  • the key features of the JavaScript language - closures, scopes, variables, Promises, EcmaScript 2018 specifics
  • browser related APIs - DOM, Events, and Ajax
  • HTML and CSS details in general and Web Components specifications
  • JavaScript's dialects - TypeScript and JSX
  • object-oriented, functional, and reactive programming paradigms
The course provides code examples, homeworks, and tasks including server side development and the NodeJS platform (streams, events, modules), and it's environments - NPM, Express, GraphQL. On the client side the following technologies are used:
  • Angular framework and the reactive streaming library RxJS
  • React and Redux
  • Web Components and technologies built on top of them
  • Polymer, Lit-HTML
  • Vue and Vuex

We respect, cover, and practice the Test-Driven Development approach for all course's exercises, including unit and end-to-end testing. For the building process we use Webpack and the community's widely used command line tools.
